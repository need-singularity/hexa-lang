# Parser @attr Mandatory ‚Äî Layer 3 of 3-Layer Law Enforcement (#4)

**Status**: design only (stage0 seed-freeze active ‚Äî see
`project_hexa_v2_circular_rebuild_20260417`). No parser edits in this session.
Wire-up deferred until hexa_v2 circular rebuild regression closes.

## Goal

Make `fn name(...) { ... }` a **parse error** unless one of the following
purity-discipline attrs precedes it:

- `@pure`           ‚Äî no side effects (contract)
- `@effect[kind]`   ‚Äî typed effect (io, net, fs, time, rand, ‚Ä¶)
- `@impure`         ‚Äî explicit side-effect opt-in (catch-all)
- `@lax`            ‚Äî **grandfather** escape hatch (migration-only,
                      pre-commit warns on count, never errors)

### Why `@lax` (not `@hexa-lax`)

`self/attrs/_naming.hexa` forbids hyphens ‚Äî single lowercase word ‚â§10 chars.
`hexa-lax` violates `naming_reason`. The spec intent is "grandfather clause";
`@lax` is the valid single-word equivalent and reads naturally as
"lax-discipline legacy fn".

### 1. `self/parser.hexa:1200` ‚Äî `fn parse_fn_decl() {`

Entry to per-fn parse. At entry, `p_pending_attrs` holds the comma-joined

```
fn parse_fn_decl() {
    // L3 purity-discipline gate (#4) ‚Äî mandatory @attr before fn
    let _purity = _parse_fn_purity_gate(p_pending_attrs)
    if _purity == "" {
        // parse error ‚Äî caller did not prepend @pure/@effect/@impure/@lax
        p_error("naked `fn` forbidden ‚Äî prepend @pure, @effect[kind], @impure, or @lax")
        // fall through so recovery can continue; gate is advisory-at-first

    ...
}
```

`_parse_fn_purity_gate(attrs: string) -> string` helper (new, in parser.hexa):

- splits `attrs` on `","`
- returns first member that matches one of {`pure`, `effect`, `impure`, `lax`}
- returns `""` if none found ‚Üí naked fn

## Grandfather Rule (`@lax`)

1. **Parser accepts** `@lax fn foo() { }` as valid ‚Äî identical to `@pure`
   in AST shape; `@lax` attr is stored in `FnDecl.value`.
2. **Pre-commit warning** (not error): count `grep -c '^@lax' *.hexa` per
   commit; print delta vs baseline (set at L3 enable commit).
   *Increasing* `@lax` count ‚Üí warn. Decreasing ‚Üí celebrate.
3. **CI gate** (future, not this layer): when `@lax` count drops to 0,
   remove `@lax` from the accept list and close the grandfather hatch.

## Non-Goals (explicit)

- **NOT** changing type system ‚Äî `@effect[io]` kind is parsed into
  `FnDecl.value` as text; effect-kind semantics is Layer 4+ work.
- **NOT** editing `codegen_c2.hexa` ‚Äî codegen ignores
  `@pure/@effect/@impure/@lax` today (all become plain C fn). Gate is
  syntactic.
- **NOT** touching interpreter ‚Äî interpreter consumes parser AST; if AST
  is built, it runs. Parser gate is sole enforcement point.
