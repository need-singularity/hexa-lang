ML 로드맵 명령어 안내를 출력한다.

아래 표를 그대로 출력하라:

```
┌────────────────────────────────────┬───────────────────────────────┐
│                명령                │             동작              │
├────────────────────────────────────┼───────────────────────────────┤
│ /ml                        │ 전체 목록 ASCII 표            │
│ /ml next                   │ 최우선 항목 자동 선택 → 구현  │
│ /ml 5                      │ #5 SSM/Mamba 구현 시작        │
│ /ml done 1                 │ #1 완료 체크 + JSON 갱신      │
│ /ml add 이름 | 설명 | 영향 │ 새 항목 추가                  │
│ /ml sync                   │ roadmap.md + CLAUDE.md 동기화 │
└────────────────────────────────────┴───────────────────────────────┘
```

SSOT: `shared/hexa-lang/ml-next-level.json`
